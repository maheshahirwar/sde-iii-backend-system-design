# Cloud (AWS) for Backend Engineers

This section focuses on **practical AWS usage**, not certification theory.

---

## Core AWS Services Covered

- EC2, IAM, VPC
- Load Balancer & Auto Scaling
- RDS (MySQL)
- S3
- CloudWatch
- API Gateway & Lambda (basics)
- SNS / SQS (eventing)

---

## Goal

- Deploy backend services on AWS
- Understand scaling & failure handling
- Map cloud services to system design components
