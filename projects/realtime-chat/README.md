# Real-Time Chat Application

## Overview
A real-time messaging system demonstrating concurrency and event-driven design.

---

## Features
- WebSocket based communication
- Typing indicators
- Online/offline presence
- Kafka-based message streaming
- Redis session cache

---

## Challenges Solved
- Message ordering
- Horizontal scaling WebSocket servers
- User presence consistency

---

## Tech Stack
- Spring Boot
- WebSocket
- Kafka
- Redis
- AWS ECS
